# Hybrid ML-Based Intrusion Detection System

Final Year / Semester Project for Information Security.

## Overview
This project implements a Hybrid Machine Learning Intrusion Detection System using:

- XGBoost Feature Selection
- Correlation Filtering
- SMOTE-ENN Balancing
- Random Forest
- XGBoost
- Stacking Ensemble Learning
- Multi-class Classification

## Dataset
CICIoT2023 Dataset

## Results
- Accuracy: 97.83%
- Weighted F1-score: 98.00%

## Files
- `infosec-finalproject.ipynb` → Full implementation
- `results/` → Output graphs and confusion matrix
